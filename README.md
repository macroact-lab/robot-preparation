## 마이캣 전원 켜기
로봇을 사용하기 위해서는 아래 그림과 같이 먼저 배 부분에 있는 배터리 씰을 제거해야 합니다.
그리고, 꼬리 부분에 있는 전원 버튼을 눌러 전원을 켜세요.<br/>
마이캣은 충전스테이션 위에 그림과 같이 모든 다리가 머리쪽으로 향하도록 올려주세요. 전원이 켜지면 눈을 감고 있는 모양이 나타납니다.</br>
